# Meridian Rooms

Static internal meeting room booking website prototype with JSON-backed sample data.

## Run

Open `index.html` directly in a browser, or serve the folder with any static file server.

The app reads its initial data from `db.json` and stores updates in `localStorage`.

## Features

- Corporate landing page
- Room discovery with filters
- Room detail and availability grid
- Booking form with validation
- Confirmation state with booking reference
